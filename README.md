# 📦 Dynamic Inventory Portal

## Overview

Dynamic Inventory Portal is a console-based inventory management system developed in C++ using Object-Oriented Programming (OOP) principles. The application helps manage inventory records efficiently by allowing users to add, delete, search, update stock levels, record sales and purchases, and generate low-stock alerts.

The system uses dynamic memory allocation to handle inventory items and file handling to maintain inventory and transaction records permanently.

---

## Features

* Add new inventory items
* Delete existing items
* Display all inventory records
* Search items by ID
* Record product sales
* Record product purchases
* Low stock alert system
* Save inventory data to files
* Store transaction history
* Dynamic inventory management using expandable arrays
